CCS selectors

A CSS selector selects the HTML elements you want to style.

  • element selector:The element selector selects HTML elements based on the element name.

  • id selector:The id selector uses the id attribute of an HTML element to select a specific element.To select an element with a specific id, write a hash (#) character.

  • class:The class selector selects HTML elements with a specific class attribute.To select elements with a specific class, write a period (.) character.

  • universal:The universal selector (*) selects all HTML elements on the page.

  • grouping selector:The grouping selector selects all the HTML elements with the same style definitions.To group selectors, separate each selector with a comma.
