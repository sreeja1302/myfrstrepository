Meta tag

< meta >: tags always go inside the < head > element, and are typically used to specify character set, page description, keywords, author of the document, and viewport settings.

character_set: Specifies the character encoding for the HTML document

content="IE=edge": we use this to avoid Compatible issues in the internet explosure

http-equiv:it can be used to refresh the page or reset the cookie

name: application-name author description generator keywords viewpoint and Specifies a name for the metadata

content="width=device-width, initial-scale=1.0: the width of the page to follow the screen-width of the device which will vary depending on the device
